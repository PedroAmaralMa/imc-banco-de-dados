# imc-banco-de-dados
Este projeto é um aplicativo desenvolvido no Android Studio para calcular o Índice de Massa Corporal (IMC), permitindo que o usuário insira seu peso e altura para obter o resultado automaticamente junto com sua classificação. Além disso, os dados informados são armazenados localmente por meio do banco de dados SQLite, possibilitando a consulta de um histórico de cálculos realizados anteriormente. O aplicativo conta com funcionalidades básicas de manipulação de dados, como inserção e consulta, e sua estrutura envolve telas de entrada, exibição de resultados e listagem das informações salvas. Trata-se de um projeto com fins educacionais, que demonstra conceitos fundamentais de desenvolvimento Android, interface gráfica e persistência de dados.
